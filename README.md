# WebMediaViewer
webpage's media file list viewer with csharp sharpdevlop

[사용법]
url 입력
이미지파일등이 포함되어 있는 태그명 입력, 속성명 입력, 이미지파일등의 확장자 선택
run 버튼 클릭
왼쪽 리스트 박스에 리스트가 나오면 해당 항목 선택하면 우측에 이미지나 동영상이 표시됨.

(이미지 파일등이 포함된 태그를 찾을려면 해당 사이트를 웹브라워저로 접속해서 소스보기 에서 참고하세요.)

주의 사항 :
이미지 파일등이 많은 사이트에 접속하면 블럭 당할수도 있으니
주의하세요....ㅜ.ㅜ;
